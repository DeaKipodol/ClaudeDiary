# Claude Session Viewer — 개발 계획

## 프로젝트 개요

Claude Code CLI 사용자를 위한 세션 기록 뷰어.
`~/.claude/projects/` 폴더를 읽어 브라우저에서 세션 목록을 보여주고,
클릭 한 번으로 터미널에서 세션을 이어서 열 수 있게 한다.

**타겟 사용자**: Claude Code CLI 설치 + 터미널에서 `claude` 명령어 사용하는 사람

- [ ] **이름 없는 세션 구분 표시**
  - 현재 54개 세션이 slug 이름(`woolly-moseying-quiche`)으로만 표시됨
  - 슬러그 세션은 흐리게 표시하거나 "(미명명)" 뱃지 추가

- [ ] **세션 삭제 방지 (고정 세션 백업)**

  - Claude Code가 세션을 지워도 복구 가능하게

- [ ] **로딩 성능 개선**

- [ ] **첫 메시지 미리보기 품질 개선**

## 기술 스택
